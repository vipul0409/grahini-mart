## 🚀 Quick Start (3 Steps)

### Step 1: Push to GitHub (5 min)
```bash
# In D:\Grahini\ folder
git init
git add .
git commit -m "Initial commit - Grahini Mart"
git remote add origin https://github.com/vipul0409/grahini-mart.git
git push -u origin main
```

### Step 2: Create Vercel Account (2 min)
1. Go to: https://vercel.com/signup
2. Click "Continue with GitHub"
3. Authorize Vercel

### Step 3: Deploy (3 min)
1. Go to: https://vercel.com/new
2. Import `vipul0409/grahini-mart`
3. Add environment variables (from deploy-commands.txt)
4. Click "Deploy"
5. Wait 2-3 minutes
6. Get your live URL! 🎉
