# ✨ Features Documentation

## 🛍️ Customer Features

### Product Browsing
- **Category Navigation**: Browse by pulses, dry fruits, spices, masala, rice, flour
- **Advanced Filters**: Filter by price, weight, category, rating
- **Search**: Real-time product search
- **Sort Options**: Popular, newest, price (low to high), price (high to low), rating
- **Product Details**: High-quality images, zoom, nutrition info, reviews
- **Weight Variants**: Multiple weight options (250g, 500g, 1kg, 2kg, 5kg)

### Shopping Cart
- **Add to Cart**: Quick add from product cards
- **Cart Management**: Update quantities, remove items
- **Cart Drawer**: Slide-out cart with live updates
- **Persistent Cart**: Cart saved across sessions
- **Coupon System**: Apply discount coupons
- **Delivery Calculation**: Auto-calculate delivery charges

### Checkout Process
- **Multi-Step Checkout**: Address → Payment → Confirmation
- **Address Management**: Save multiple addresses
- **Payment Options**: 
  - Razorpay (UPI, Cards, Net Banking, Wallets)
  - Cash on Delivery (COD)
- **Order Summary**: Clear breakdown of costs
- **Order Confirmation**: Email and SMS notifications

### User Account
- **Profile Management**: Update personal information
- **Order History**: View all past orders
- **Order Tracking**: Real-time order status
- **Reorder**: Quick reorder from history
- **Wishlist**: Save favorite products
- **Saved Addresses**: Manage delivery addresses
- **Loyalty Points**: Earn and redeem points

### Additional Features
- **WhatsApp Integration**: Direct order via WhatsApp
- **Product Reviews**: Read and write reviews
- **Ratings**: Star ratings for products
- **Notifications**: Push notifications for offers
- **Referral Program**: Refer friends and earn rewards
- **Multi-Language**: Support for multiple languages
- **Dark Mode**: Toggle dark/light theme
