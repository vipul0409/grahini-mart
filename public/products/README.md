# 📸 Product Images Folder

## Add Your Product Images Here

Place your product images in this folder with these exact names:

1. **toor-dal.jpg** - Toor Dal (Pulses) image
2. **cashews.jpg** - Cashews (Kaju) image
3. **almonds.jpg** - Almonds (Badam) image
4. **red-chilli-powder.jpg** - Red Chilli Powder image
5. **turmeric-powder.jpg** - Turmeric Powder image
6. **garam-masala.jpg** - Garam Masala image

## Image Requirements

- **Format:** JPG or PNG
- **Size:** Minimum 800x800 pixels
- **Quality:** High resolution, clear product visibility
- **Background:** Clean, professional

## After Adding Images

1. Make sure all 6 images are in this folder
2. Check filenames match exactly (case-sensitive)
3. Restart your dev server: `npm run dev`
4. Visit homepage to see your images!
