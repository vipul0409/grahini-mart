# 🎨 Banner Images Folder

## Add Your Hero Slider Banner Images Here

Place your banner images in this folder with these exact names:

### Desktop Banners (Landscape - 1200x500px recommended)
1. **dry-fruits-banner.jpg** - Dry fruits banner (cashews, almonds, etc.)
2. **masala-banner.jpg** - Masala powders banner (spices, turmeric, chilli)
3. **pulses-banner.jpg** - Pulses/Dal banner (toor dal, moong dal, etc.)

### Mobile Banners (Portrait/Square - 800x600px recommended)
4. **dry-fruits-banner-mobile.jpg** - Dry fruits mobile banner
5. **masala-banner-mobile.jpg** - Masala powders mobile banner
6. **pulses-banner-mobile.jpg** - Pulses/Dal mobile banner

## Image Requirements

- **Format:** JPG or PNG
- **Desktop Size:** 1200x500 pixels (landscape, wide)
- **Mobile Size:** 800x600 pixels (more square/portrait)
- **Quality:** High resolution, vibrant colors
- **Style:** Eye-catching, promotional, with products visible

## Design Tips

- Use bright, attractive colors
- Show multiple products in the banner
- Keep text area clear (left or center)
- Make sure products are clearly visible
- Use professional photography

## After Adding Images

1. Make sure all 6 images are in this folder
2. Check filenames match exactly (case-sensitive)
3. Restart your dev server: `npm run dev`
4. Visit homepage to see your slider banners!
